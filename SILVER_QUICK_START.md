# Silver Tier - Quick Start Guide

## Complete Flow for LinkedIn Automation

### Prerequisites

```bash
# Install Python dependencies
pip install -r requirements.txt

# Install Playwright browsers
playwright install

# Install Node.js dependencies for MCP servers
cd mcp-servers/email-mcp && npm install
cd ../linkedin-mcp && npm install
```

---

## Step-by-Step Flow

### Step 1: Login to LinkedIn (One-time)

```bash
python scripts/linkedin_login.py
```

**What happens:**
- Browser opens (visible)
- Login to LinkedIn with your credentials
- Session saved to `linkedin_session/` folder
- Close browser when done

---

### Step 2: Start LinkedIn Watcher

```bash
python scripts/linkedin_watcher.py
```

**What happens:**
- Watcher runs in background (checks every 5 minutes)
- Creates sample LinkedIn opportunity (first run demo)
- Action file created in `/Needs_Action/`
- Ready for Qwen AI to process

---

### Step 3: Process with Qwen AI

```bash
# In a new terminal, run Qwen Code
qwen --prompt "Process the LinkedIn item in /Needs_Action/ folder. Follow the Company Handbook rules and create a post draft."
```

**What happens:**
- Qwen reads the action file
- Reviews Company Handbook for rules
- Creates post draft in `/Plans/` folder
- Creates approval request in `/Pending_Approval/`

---

### Step 4: Approve and Post

```bash
# Review the approval request
cat personal-ai-employee/Pending_Approval/APPROVAL_LINKEDIN_POST_*.md

# If approved, run the post script
python scripts/linkedin_post.py
```

**What happens:**
- Browser opens with your saved LinkedIn session
- Qwen AI generated content is entered
- You review and click "Post" button
- Result saved to `/Done/` folder

## Qwen AI Integration

### Configure Qwen Code

Add this to your Qwen Code configuration:

```markdown
# System Prompt for AI Employee

You are my AI Employee, working 24/7 to manage my business.

## Your Tools:
- **Email MCP**: Send emails, create drafts
- **LinkedIn MCP**: Create posts, send messages, connect
- **File System**: Read/write to Obsidian vault

## Rules (from Company Handbook):
1. Always be transparent - log every action
2. Require approval for: payments >$50, posts, new contacts
3. Be proactive - identify opportunities
4. Document everything

## Workflow:
1. Read action files from /Needs_Action/
2. Process according to Company Handbook
3. Create plans in /Plans/
4. Create approval requests in /Pending_Approval/
5. When approved, execute via MCP
6. Move completed to /Done/
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Not logged in" | Run `python scripts/linkedin_login.py` again |
| No action file created | Check watcher logs: `logs/linkedin_watcher.log` |
| Qwen not processing | Ensure Qwen Code is installed and in PATH |
| Post fails | Check LinkedIn session, may need to re-login |
